# VLA-and-RLHF-for-Robot-Motion-Planning

This project is proposed to develop a language
driven motion planning framework that integrates vision
language action (VLA) and learning from human feedback
(RLHF) reasoning for single-robot or multi-robot navigation
in dynamic environments. The framework leverages multimodal
perception, including Bird’s-Eye View (BEV) maps and LiDAR
based obstacle detection, to generate context-aware navigation
directives through VLA models. A hierarchical architecture
combines high-level semantic reasoning with low-level policy
optimization, where RLHF aligns robot behaviors with human
preferences. The system is validated through simulation and
real-world experiments, demonstrating improved success rates,
reduced unsafe interactions, and interpretable decision-making
compared to baseline VLA models. Results indicate that in
tegrating VLA reasoning with RLHF enables socially compli
ant navigation under decentralized, non-communicating crowd
conditions, bridging the gap between semantic understanding
and reactive control. Future work will extend this approach
to multi-robot coordination and incorporate human-in-the-loop
feedback for richer social alignment.
